# Cases
В данном репозитории содержатся кейсы, которые были выполнены в рамках различных кейс-чемпионатах, хакатонах и прочих соревновательных мероприятиях

## Кейсы:

| №  | Название и ссылка                              | Описание                          |
|:---|:-----------------------------------------------|:----------------------------------|
| 1. | [Разработка стратегии продвижения P&G в эпоху нейросетей](p_and_g/)                | В последние годы меняется процесс выбора товаров. Для этого пользователи используют поиск с генеративным движком. Таким образом, на окончательный выбор пользователя влияет результат суммаризации нейросетей (т.е. ее рекомендации на внешние источники). Кейс направлен на разработку комплексной стратегии продвижения брендов P&G в системах генеративного поиска для обеспечения лидирующих позиций в новой поисковой экономике. |
